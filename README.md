# ZiDAG
Directed Graphical Models and Causal Discovery for Zero-Inflated Data based on Hurdle conditionals. Implementation for "Directed Graphical Models and Causal Discovery for Zero-Inflated Data" (under review).

The DESCRIPTION and README will be updated upon publication of the paper.

# Installation from GitHub
```R
install.packages(c("devtools", "knitr"))
devtools::install_github("sqyu/ZiDAG")
```
